# Lottery Token NFT System

Solidity smart contract project for an on-chain lottery flow built around token and NFT-style mechanics.

## Overview

This repository experiments with lottery logic implemented directly in Solidity. The contract is structured as a compact learning project for combining token-based interactions and NFT-related ideas in a single system.

## Contract

- `lottery.sol`: main lottery contract

## Usage

Compile and test the contract with Remix or a compatible Solidity workflow.
