# Lottery Token NFT System

[![Solidity](https://img.shields.io/badge/Solidity-0.8-363636?logo=solidity)](https://soliditylang.org/)
[![Ethereum](https://img.shields.io/badge/Ethereum-EVM-3C3C3D?logo=ethereum)](https://ethereum.org/)

An on-chain lottery system in Solidity combining ERC-20 token mechanics with NFT-style ticket representation for provably fair reward distribution and transparent draw mechanics.

## Features

- **On-chain lottery** — fully transparent draw mechanism on the blockchain
- **Token-based tickets** — purchase lottery entries using token mechanics
- **NFT-style representation** — each ticket represented as a unique digital asset
- **Provably fair draws** — deterministic, verifiable winner selection
- **Reward distribution** — automatic prize allocation to winning tickets
- **Deploy-ready** — Solidity 0.8 contract compatible with any EVM network

## Tech Stack

- **Solidity ^0.8.0** — Smart contract language
- **Ethereum EVM** — Compatible with any EVM network
